# SKN14-EDA-5Team

## 팀 소개

SKN5 팀 프로젝트: **EDA**를 통한 신규 OTT 서비스에 관한 인사이트 (넷플릭스 데이터분석)
저희 SKN14기 5팀은 신규 OTT서비스 런칭을 가정하여 시장에서의 경쟁자인 넷플릭스의 데이터를 분석하여 인사이트를 도출하였습니다. 이 프로젝트는 넷플릭스 구독자 연령대별, 국가별, 장르별로 분석하고 및 시각화 자료를 제공합니다.

### 팀원소개 및 역할분담
* 승철님 : 넷플릭스 데이터 수집, 전처리, 세분화, 분석 및 국가별 서비스 확장 전략 수립

* 정규님 : 연령대별/국가별/장르별 데이터 분석 

* 유나님 : 구독 요금제의 특징에 관한 데이터 분석

* 은선님: 구독자 중 잠재적 이탈자에 관한 데이터 분석 

* 유진님: 연령대별/국가별/장르별 데이터 분석

💡 핵심 목표
핵심 목표는 넷플릭스의 사례를 참고하여 OTT 서비스의 성공적인 전략 및 인사이트를 제공하는 것입니다. 

🏆 프로젝트 목표 및 배경
# 🎬 신규 OTT 서비스 전략 분석 프로젝트

## 📋 프로젝트 개요

### 배경
- 글로벌 OTT 시장에서 넷플릭스가 선도적 위치를 차지하고 있음
- 경쟁력 있는 신규 OTT 서비스 런칭을 위한 데이터 기반 전략 수립 필요
- 넷플릭스의 사용자 데이터 분석을 통한 시장 진입 전략 도출

### 목표
- 넷플릭스 사용자 행동 패턴 분석
- 수익성 높은 구독 모델 설계
- 콘텐츠 전략 수립
- 사용자 이탈 방지 전략 수립

  
## 📌 문제 제기
- 구독 유형을 Premium-Standard-Basic으로 피라미드화한 요금제 변경의 효과 검증 필요
- 요금제 변경이 사용자 행태에 미친 영향 분석
- 구독 유형별 차별화 요소 발굴

### 1. 구독 패턴
- 구독 유형별 평균 시청 시간:
  - Basic: 498시간
  - Standard: 502시간
  - Premium: 504시간
- 구독료 차이 대비 시청 시간 차이가 크지 않음

### 1.1 구독 유형별 심층 분석

#### 1.1.1 장르 선호도
| 구독 유형 | 선호 장르 1 | 선호 장르 2 | 선호 장르 3 |
|---------|------------|------------|------------|
| Premium | Action (35%) | Documentary (25%) | Thriller (20%) |
| Standard | Drama (30%) | Comedy (25%) | Action (20%) |
| Basic | Drama (40%) | Comedy (30%) | Romance (15%) |

#### 1.1.2 시청 패턴
- **Premium**
  - 주중/주말 고른 시청 분포
  - 프라임타임(20-23시) 집중

- **Standard**
  - 주말 시청 비중 60%
  - 저녁시간(18-22시) 집중

- **Basic**
  - 주말 시청 비중 70%
  - 저녁시간(19-21시) 집중

## 💡 전략적 제언

### 1. 구독 모델 최적화
- 차별화된 3단계 구독 모델 설계
- 각 등급별 명확한 가치 제안
- 사용 시간에 따른 인센티브 제도 도입

### 2. 콘텐츠 전략
- Drama, Comedy 중심의 초기 라인업 구성
- 국가별 선호 장르를 고려한 현지화 전략
- 연령대별 맞춤형 콘텐츠 추천 시스템 도입

### 3. 이탈 방지 전략
- 사용자 활동 모니터링 시스템 구축
- 저활동 사용자 대상 맞춤형 프로모션
- 구독 등급별 차별화된 리텐션 프로그램 운영

## 📈 기대 효과
- 안정적인 구독자 기반 확보
- 높은 사용자 만족도 달성
- 효율적인 콘텐츠 투자
- 지속 가능한 수익 모델 구축

## 🎯 핵심 차별화 전략

### 1. 콘텐츠 '스토리텔링' 강화
- **데이터 인사이트**: 
  - Premium 사용자들이 선호하는 장르: 로맨스(25%), 드라마(22%), 액션(18%)
  - 장르별 평균 시청 시간이 490-510시간으로 비슷 → 콘텐츠 퀄리티가 관건

- **전략 방향**:
  - 글로벌 시장 타겟 오리지널 콘텐츠 제작
  - 미국/영국 시장 특화 콘텐츠 우선 개발
  - 스토리 중심의 시리즈물 강화
  
### 2. 시즈널 콘텐츠 전략
- **데이터 인사이트**:
  - 휴가 시즌 시청 시간 증가
  - 국가별 성수기 패턴 상이

- **전략 방향**:
  - 휴가 시즌 맞춤형 컨텐츠 릴리즈
  - 시즌별 특화 콘텐츠 기획
  - 국가별 주요 휴가 시즌 타겟팅
  - 시즌 특화 프로모션 연계

### 3. 스마트 요금제 설계
- **데이터 인사이트**:
  - 구독 유형별 고른 분포도
  - Premium 구독자의 높은 이탈률
  - 시청 시간대비 요금제 차별성 부족

- **전략 방향**:
  - 사용량 기반 요금제 도입
  - 프리미엄 전용 컨텐츠 강화
  - 장기 구독자 혜택 제도
  - 계절성 할인 프로모션

### 4. 전략적 릴리즈 일정
- **데이터 인사이트**:
  - 요일별/시간대별 시청 패턴
  - 국가별 선호 시청 시간대

- **전략 방향**:
  - 빅데이터 기반 최적 공개 시점 선정
  - 글로벌 시차 고려한 릴리즈
  - 시리즈물 연속성 확보
  - 경쟁사 일정 회피 전략
